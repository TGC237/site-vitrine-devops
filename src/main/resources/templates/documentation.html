Documentation Projet

Référentiel technique et procédures opérationnelles.

1. Initialisation

Le projet utilise Spring Boot 3.2.0 et Java 17. La gestion des dépendances est assurée par Maven.

2. Sécurité (DevSecOps)

Aucun mot de passe en clair. Utilisation des Jenkins Credentials et tokens d'accès. Le trafic est chiffré TLS/SSL de bout en bout.

3. Maintenance

Les logs sont accessibles via docker logs -f site-vitrine. Le redémarrage est automatique en cas d'échec (policy: always).
